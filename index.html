Hello

After studying arts, earning a degree in literature, and working in the non-profit sector, I discovered a passion for programming. I started learning on my own, first becoming a frontend web developer, and later moving into game development.

Most of my skills were built through self-directed learning, free online courses, and personal projects. More recently, I completed the Online Master in Game Programming at Digital Bros Game Academy and worked on a commercial mobile game set to be released soon.

I have experience with both 2D and 3D games using Unity and C#, and Unreal Engine 5 and C++ (and Blueprints). I’m especially passionate about gameplay programming and interested in learning more about tool development.

I enjoy the complexity of game programming: the challenge of building systems that work smoothly together, balancing performance optimization with clean, readable code. I love working independently as well as in a team, embracing challenges, and constantly learning new things.
